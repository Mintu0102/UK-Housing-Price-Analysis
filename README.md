# UK Housing Price Analysis

## Project Overview

This project analyzes UK property transactions using Python and Pandas. The objective is to identify pricing trends, property type performance, geographic differences, and business insights from housing transaction data.

---

## Dataset

- UK Housing Transactions (2026)
- Approximately 195,000 property transactions

---

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

---

## Project Workflow

- Data Cleaning
- Handling Missing Values
- Feature Engineering
- Exploratory Data Analysis
- GroupBy Analysis
- Pivot Tables
- Data Visualization
- Business Insights

---

## Key Questions Answered

- Which counties have the highest average house prices?
- Which cities have the most transactions?
- Which property type generates the highest sales?
- Which months have the highest average selling prices?
- How are luxury properties distributed?

---

## Visualizations

- Average Price by County
- Property Type Sales
- Monthly Price Trends

---

## Business Insights

- Detached properties generate the highest sales value.
- Major cities account for most property transactions.
- Luxury properties represent a small percentage of sales but contribute significantly to total market value.
- House prices differ considerably across counties.
- Monthly price trends reveal changes in market activity.
